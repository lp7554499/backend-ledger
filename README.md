# Backend Ledger System

A simple **Backend Ledger System** built with **Node.js, Express, and MongoDB** to manage bank-like transactions such as deposits, withdrawals, and transaction history.

---

## 🚀 Features

* Create user accounts
* Deposit money
* Withdraw money
* Store transaction history
* View account balance
* REST API for ledger operations

---

## 🛠 Tech Stack

* **Node.js**
* **Express.js**
* **MongoDB**
* **Nodemon**

---

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/your-username/backend-ledger.git
```

Go to project folder:

```
cd backend-ledger
```

Install dependencies:

```
npm install
```

---

## ▶️ Run the Project

Start the development server:

```
npm run dev
```

Server will run on:

```
http://localhost:3000
```

---

## 📡 Example API Endpoints

### Create Account

```
POST /api/account
```

### Deposit Money

```
POST /api/deposit
```

### Withdraw Money

```
POST /api/withdraw
```

### Transaction History

```
GET /api/transactions
```
